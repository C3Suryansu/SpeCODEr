<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Spectrum-CETB/SpeCODEr.git
   ```
2. Install Python Libraries
   ```sh
    pip install flask
    pip install subprocess
    ```
